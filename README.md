## finance-bi-analytics

**End-to-End Finance Analytics Project** using Python, Power BI, and Tableau.

### Overview
This project demonstrates a complete analytics workflow: from raw financial data ingestion, through cleaning and transformation in Python, to delivering business insights via professional dashboards in both Power BI and Tableau.

### Key Features (Planned)
- Python data pipeline for financial data
- Data modeling best practices
- Power BI semantic model + advanced DAX
- Tableau workbook with calculated fields and LODs
- Documentation of data lineage and business logic
- Comparison of Power BI vs Tableau approaches

### Tech Stack
- Python (Pandas/Polars)
- Power BI Desktop
- Tableau Desktop/Public
- SQL (for data modeling)
- DAX
